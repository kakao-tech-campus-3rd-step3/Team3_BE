팀 매치·용병·리뷰·토너먼트까지
한 곳에서 끝내요.

대학 이메일 인증으로 신뢰를 더하고, 팀 매치 생성/참여, 용병 모집/수락, 경기 후 리뷰까지 모두 연결합니다. 500ms 이내 반응 속도로 빠르게.

✓
대학교 이메일 인증ac.kr 기반 본인 확인으로 안전한 매칭

핵심 기능

테크 스펙 우선순위: 인증 → 매치 관리 → 용병 매칭 (그 다음: 팀 관리, 토너먼트)

인증

대학교 이메일 기반 가입

학교 이메일(ac.kr) 인증으로 동아리·팀 구성원만 모여 안전합니다. JWT 기반 로그인/토큰 갱신 지원.

매치

매치 생성·참여

날짜·시간·구장·실력대를 설정해 매치를 열고, 조건 필터로 참여 가능한 매치를 빠르게 찾습니다.

용병

용병 모집·수락

부족 인원을 포지션별로 모집하고, 지원자를 평판/실력으로 정렬해 수락·거절을 간편하게.

리뷰

경기 후 팀/용병 리뷰

시간 준수·매너·실력 등을 별점/코멘트로 남겨 다음 매칭의 참고 지표로 활용합니다.

팀 관리

팀 구성원/통계

팀원 목록, 포지션, 승률·전적 요약을 제공. 팀 운영에 필요한 핵심 정보가 한 화면에.

토너먼트

대회 참가/브라켓

16팀 모이면 자동 대진표 생성. 진행 상태와 결과 관리를 지원합니다.

이용 흐름

가입부터 리뷰까지, 5단계로 끝!

1. 가입대학 이메일 인증으로 안전하게 시작
2. 팀 등록팀 생성 및 멤버 초대
3. 매치매치 생성 또는 조건 맞는 매치 참여
4. 용병부족 인원은 포지션별로 빠르게 모집
5. 리뷰경기 후 팀/용병 리뷰로 신뢰도 축적

안정적인 기술 스택

Spring Boot · MySQL(RDS) · AWS(EC2·S3) · Redis · React Native + Expo · TypeScript · Kakao Map API

빠른 응답

목록 조회·필터·매치 생성/수락 500ms 목표로 쾌적한 경험.

보안/프라이버시

JWT 인증, 비밀번호 SHA-256 저장, 대학 이메일 검증으로 안전.

지도/구장

주변 구장 검색, 가격/시설 정보, 길찾기 안내(계획)까지.

자주 묻는 질문

누가 사용할 수 있나요?
강원도 대학생을 핵심 타깃으로 시작하며, 대학 이메일(ac.kr) 인증으로 가입합니다.
용병은 어떻게 모집하나요?
팀 생성 페이지에서 부족 포지션·인원·시간을 지정해 공고를 올리면, 지원자를 리뷰/실력 기준으로 정렬해 수락할 수 있습니다.
리뷰는 필수인가요?
리뷰는 선택 사항이나, 추후 매칭의 신뢰도를 높이기 위해 경기가 끝난 후 팀/용병 리뷰를 권장합니다.
